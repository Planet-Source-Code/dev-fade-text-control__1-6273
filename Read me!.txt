2.	Using Dev Fade OCX:
	------------------

  Running Dev Fade OCX:
  -------------

To run Dev Fade OCX run FadeOCX.vbg


  The Properties:
  --------------

Backcolor:	Sets the background colour of the control.
Example:	Fade1.Backcolor = VBButtonFace


Caption:	Sets the caption of the faded text.
Example:	Fade1.Caption = "Dev Fade OCX"


Color1:		Sets the first colour of the text fade
Example:	Fade1.Color1 = VBRed


Color2:		Sets the last colour of the text fade
Example:	Fade1.Color2 = VBBlack


Font:		Sets the font of the control.
Example:	Fade1.font = Arial Black
